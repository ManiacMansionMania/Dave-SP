# Maniac Mansion Mania Dave-SP #
## Renovation ##
This is a reworked version of the Dave starterpack with an altered floor plan. The goal of this version is to make the room layout more logical (e.g. no need to go through the bedroom to enter the kitchen).

This is work a process. Contents can change anytime.

The starterpack is only avaiable in german. An english translation may be possible after the changes are complete.
